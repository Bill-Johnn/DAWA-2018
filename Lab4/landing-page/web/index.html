    ¿Como Utilizar?

    Pagina Web

    Igrese a la pagian web para poder registrase, una vez registrado tenemos dos modos de ususario, uno cliente y otro vendedor.

    Busque sus productos

    Si usted esta como modo comprador tendra una pagina en el cual podra ver atraves de un mapa donde estan los productos de reciclaje.

    Publique productos

    Si usted esta en modo vendedor podra publicar los podructos de reciclaje de modo facil, solo indicando su lugar donde recogera el producto.
